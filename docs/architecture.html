ShellBot v0.2 Architecture

Bash-native AI agent — Function Calling with @tool annotations → OpenAI JSON Schema

Two Execution Modes

  • • react.sh — Single-turn function calling (default)
  • • loop.sh — Multi-turn autonomous loop (--loop)
  • • Both share api.sh, tools.sh, memory.sh
  • • context.sh only used in loop mode

14 Tool Implementations

  • • Shell: run_shell, read/write/search_files
  • • Web: search_web, read_webpage
  • • KB: search/read_rhkb (Red Hat SSO)
  • • Memory: save/search_memory → SQLite3

Schema Generation

  • • @tool/@param annotations in Bash comments
  • • tools_schema.sh → OpenAI JSON Schema
  • • Zero Python dependency for core loop
  • • Lazy-loaded compressor (LLM summarize)
